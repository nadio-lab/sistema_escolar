# 🏫 Sistema de Gestão Escolar - Colégio Visão do Futuro

Este é um sistema completo de gestão escolar desenvolvido em **PHP** e **MySQL**, focado na automatização de processos académicos e administrativos, permitindo uma comunicação fluida entre a direção, professores e alunos.

## 🚀 Funcionalidades Principais

### 👨‍🎓 Portal do Aluno
* **Dashboard Personalizado:** Saudação dinâmica e resumo de atividades.
* **Desempenho Académico:** Visualização de notas e médias com cálculo automático de estado (Aprovado, Exame, Recurso).
* **Gráfico de Evolução:** Acompanhamento visual do desempenho nos três trimestres via Chart.js.
* **Calendário de Propinas:** Consulta de meses pagos e pendentes com geração de **Recibos em PDF**.
* **Entrega de Tarefas:** Upload de trabalhos diretamente para os professores.
* **Cartão de Estudante Digital:** Identificação personalizada com foto e dados do aluno.

### 👨‍🏫 Painel do Professor
* **Gestão de Turmas:** Listagem de alunos vinculados por disciplina.
* **Lançamento de Notas:** Caderneta digital para preenchimento de notas trimestrais.
* **Receção de Trabalhos:** Filtro e download de ficheiros enviados pelos alunos.
* **Diário de Classe:** Registo de sumários e conteúdos lecionados.

### ⚙️ Administração
* **Gestão de Matrículas:** Cadastro de alunos com configuração de login/senha.
* **Configurações do Colégio:** Personalização de logótipo, cores e dados institucionais.
* **Mural de Atividades:** Publicação de eventos e avisos gerais.

## 🛠️ Tecnologias Utilizadas

* **Backend:** PHP 8.0+
* **Base de Dados:** MySQL (MariaDB)
* **Frontend:** HTML5, CSS3 (Variáveis modernas), JavaScript (ES6)
* **Bibliotecas:** [Chart.js](https://www.chartjs.org) (Gráficos), FontAwesome (Ícones)
* **Servidor:** XAMPP / Apache
